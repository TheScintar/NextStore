## Try it on ([https://polite-sfogliatella-58ade3.netlify.app/](https://storenext.netlify.app/))

Online store made using Next.js & React.js & Firebase with the ability to create an account and add products to cart

## Features

- Authentication
  - Sign up
  - Login
  - Logout
- Filters
- Cart Management
- User profile page
- Single Product Page.
- 
## Tech Stack Used
- NextJs
- ReactJS
- Firebase Auth
- Firebase Database
